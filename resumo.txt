O algoritmo é:

- Quando o usuário clicar em pesquisar, traduzir o termo de pesquisa (nome do personagem) para inglês.
- Com o nome do personagem em inglês, fazer uma requisição à API da Marvel.
- Com os dados em português, exibi-los em tela.

Ao todo, são x requisições:
1. Traduzir o termo de pesquisa para inglês.
2. Pesquisar os dados do personagem com o termo de pesquisa.
3. Pesquisar pelo personagem.
4. Traduzir os dados do personagem.
